# Deprecations

## Overview

This page lists API deprecations and recommended replacements.

## Upcoming deprecations

### 2026-06-03: Reusable prompts

On June 3, 2026, reusable prompt objects were deprecated.

| Date | Update |
| --- | --- |
| June 3, 2026 | Deprecation announced. |
| Nov 30, 2026 | The `v1/prompts` API is scheduled to shut down. |

To migrate, move reusable prompt content into application code.

### 2026-06-02: GPT Image model deprecations

Older GPT Image models are deprecated and scheduled for removal.

| Shutdown date | Model / system | Recommended replacement |
| --- | --- | --- |
| Dec 1, 2026 | `gpt-image-1-mini` | `gpt-image-2` |
